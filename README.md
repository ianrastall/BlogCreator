# BlogCreator

BlogCreator is a separate Windows desktop publishing application for the Fooliosity website.

It will be built with C#, .NET 10, WinUI 3, MVVM, CommunityToolkit.Mvvm, and a dark-mode interface.

Project boundary:

- `ianrastall/BlogCreator` is the desktop application.
- `ianrastall/ianrastall.github.io` is the public website source.
- `ianrastall/MDEdit` is an independent Markdown editor and must remain untouched.

The application may reuse architecture and ideas from MDEdit, but it should not be developed inside the MDEdit repository.
